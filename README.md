# Proyecto Final - CV Web con Docker

Aplicación web de Curriculum Vitae desarrollada con:

- Frontend: React
- Backend: Node.js + Express
- Base de datos: MySQL
- Contenedores: Docker y Docker Compose

---

# Tecnologías utilizadas

- React
- Node.js
- Express
- MySQL 8
- Docker
- Docker Compose

---

# Estructura del proyecto

```txt
practica-final-cv/
│
├── backend/
├── frontend/
├── database/
│   └── init.sql
├── docker-compose.yml
└── README.md
```

---

# Funcionalidades

- Visualización de Curriculum Vitae
- Backend conectado a MySQL
- Frontend consumiendo API REST
- Base de datos creada automáticamente
- Contenedores Docker independientes
- Orquestación completa con Docker Compose

---

# Ejecución del proyecto

## 1. Clonar repositorio

```bash
git clone https://github.com/Yagamesa/practica-final-cv.git
```

---

## 2. Entrar al proyecto

```bash
cd practica-final-cv
```

---

## 3. Ejecutar Docker Compose

```bash
docker compose up -d --build
```
